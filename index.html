About MedTracker
Med
TRACKER DEMO

MedTracker is your personal home medicine inventory management system. Track expiry dates, manage dosage schedules, and never run out of essential medicines.

Designed following Shneiderman's 8 Golden Rules of Interface Design to ensure a consistent, efficient, and user-friendly experience.

Version 1.0 · HCI Group 8 Project 2026
🏆 Shneiderman's 8 Golden Rules of Interface Design ▼
1
Strive for Consistency — Uniform button styles, colors, typography, and navigation across all pages. The blue palette and Nunito font are used throughout.
2
Seek Universal Usability — Color-coded status badges, readable font sizes, and clear labels ensure users of all skill levels can navigate effectively.
3
Offer Informative Feedback — Toast notifications confirm every action (add, delete, save). Status badges (EXPIRED, EXPIRING, IN-DATE) give immediate visual feedback.
4
Design Dialogs to Yield Closure — Add medicine modal has clear Cancel and Save Record actions. Success toast confirms completion.
5
Prevent Errors — Form validation highlights empty required fields. Delete confirmation dialogs prevent accidental data loss.
6
Permit Easy Reversal of Actions — Cancel buttons in all modals. Confirmation step before irreversible actions (delete, clear data).
7
Keep Users in Control — Sidebar navigation always visible. Tabs let users filter their view. Filter and search give full control over displayed data.
8
Reduce Short-Term Memory Load — Status shown directly on each item. Dashboard summaries reduce need to remember counts. Auto-fill in medicine name field reduces typing effort.
